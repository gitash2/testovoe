# Startup instruction

git clone https://github.com/gitash2/testovoe.git

Запустить либо в IDE, либо через 

mvn clean package

java -jar target/testovoe-0.0.1-SNAPSHOT.jar

После запуска Swagger UI будет доступен по адресу:

http://localhost:8080/swagger-ui/index.html

В корне проекта лежит файл с тестовыми данными.
